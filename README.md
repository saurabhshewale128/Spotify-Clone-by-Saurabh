# 🎵 Spotify Clone

A responsive Spotify Clone built using **HTML** and **CSS**. This project recreates the Spotify web player interface with a modern and user-friendly design.

## 🚀 Features
- Responsive UI
- Sidebar Navigation
- Music Player Layout
- Playlist & Album Cards
- Spotify-inspired Design

## 🛠️ Technologies Used
- HTML5
- CSS3
- Font Awesome
- Google Fonts
